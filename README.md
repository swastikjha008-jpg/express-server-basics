# Express Server Basics 🚀

A simple backend server built using Node.js and Express.

## ⚙️ Features
- GET / → Check server
- GET /users → Get all users
- POST /users → Add user
- DELETE /users/:id → Delete user

## 🛠️ Tech Stack
- Node.js
- Express.js

## ▶️ How to Run

```bash
